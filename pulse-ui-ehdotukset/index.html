Käyttöliittymäsuunnitelma

PULSE mobiilinäkymät template-elementeillä.

Tämä versio näyttää varsinaiset sovellusnäkymät, ei ominaisuuden myyntiesittelyä. Jokainen ruutu käyttää mobile trio -templateen rakennettuja elementtejä: app chrome, tabs, report cards, list items, badges, forms, dialog, progress ja bottom navigation.

A
Osallistujan sessionäkymäPULSE widget sijoitetaan ohjelmakortin sisään ja säilyttää yhden napautuksen reaktion.
B
Syykeräyksen bottom sheetKevyt dialogi käyttää template-lomakekomponentteja ja sulkee raskaan kyselytunnelman pois.
C
Järjestäjän signaalinäkymäRaportti käyttää report cardeja, list itemeitä, progress-palkkeja ja toast-tyylistä insightia.
