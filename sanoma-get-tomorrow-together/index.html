Case reference · Sanoma Get Tomorrow — Together · 23 April 2026

One event. Personal to every attendee.

For Sanoma Get Tomorrow — Together, Lataamo’s event app added an AI layer on top of the live programme: every participant built a profile, received relevant people matches, and continued after the event in a personal hub with tailored insights and tools to translate the event’s messages for different colleagues.

The value: the event no longer ends at inspiration. The personal hub keeps each attendee’s feed alive and lets them create audience-specific insights — for example for sales — so the event’s messages travel further inside the organization.
